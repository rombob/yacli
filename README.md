Prerequisites
-------------
You will need the following:

- Ruby 1.9+
- Bundler

## Installation

Add this line to your application's Gemfile:

    gem 'yacli'

And then execute:

    $ bundle install

Or install it yourself as:

    $ gem install yacli

## Usage

    $ yacli help [COMMAND]                                          # Describe expanded available commands or one specific command
